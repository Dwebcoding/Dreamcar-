# Changelog - DreamCar

All significant changes to the project will be documented in this file.

The format is based on Keep a Changelog (https://keepachangelog.com/en/1.0.0/),
and this project adheres to Semantic Versioning (https://semver.org/).

## [1.0.0] - 2026-01-28

### Added
- Complete responsive website design
- Curtain animation on load
- Mobile hamburger menu
- Interactive quote form
- Dynamic car brand/model selector
- CarQuery API import script
- Complete documentation (README, CONTRIBUTING)
- GitHub configuration (.gitignore, LICENSE)
- JavaScript performance optimizations
- Scroll reveal animations
- Smart header with scroll behavior
- Pages: Home, Services, About, Quote, Contact
- Legal pages: Privacy, Cookie Policy, Terms of Service
- Sitemap

### Changed
- Complete JavaScript refactoring with ES6+ patterns
- Modular IIFE structure for all scripts
- JSDoc for function documentation
- Improved quote form UX
- Optimized package.json configuration

### Optimized
- Scroll performance with requestAnimationFrame
- DOM manipulation with DocumentFragment
- Passive event listeners for scroll
- Reduced code duplication
- Improved client-side validation

### Technical
- Node.js 14+ as minimum requirement
- Modern browser support (last 2 versions)
- CarQuery API integration
- Mobile-first responsive design
- Improved accessibility with ARIA

- **Fixed** - bug fixes

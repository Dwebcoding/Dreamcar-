## Setup Veloce

```bash
# 1. Clone
git clone https://github.com/yourusername/dreamcar-site.git

# 2. Apri nel browser
open index.html  # macOS
start index.html # Windows

# 3. Oppure con server
python -m http.server 8000
```

Poi apri: http://localhost:8000
